当前重点

源码公开

包含启动器主体、UI、动画、下载、Minecraft 启动、资源管理、PCLCS 与 MeloongCore 等代码。

构建公开

使用 GitHub Actions 在 Windows 环境中执行 Release 构建，并上传可下载产物。

联机内部化

PCL2 默认使用已移植到 VB/.NET 的内部 Taowa 联机实现，不再携带或启动 terracotta.exe。

主题恢复

已恢复主题解锁与原版主题调色逻辑，便于独立使用与继续维护。

版本分支

主分支用于继续维护完全开源正式版，参考分支保留旧正式版和 Taowa 测试版快照，方便对照恢复过程和行为差异。
